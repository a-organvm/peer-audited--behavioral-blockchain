# GTM War Board

This is the compressed operating view for the first 30 days of go-to-market.

Use it in weekly or twice-weekly reviews. If a lane cannot point to proof, it is not done.

## Scope Lock

All lanes operate inside this boundary:

- `No-Contact recovery`
- `iOS` private beta
- `Test-Money Pilot`
- `US allowlist`

Do not expand into generic habit marketing, real-money positioning, or broad platform storytelling.

## Current Done Criteria

Nothing counts as done unless the proof exists.

- copy is done only when the approved source text is fixed
- outreach is done only when the sent log exists
- page work is done only when the route is live or previewable
- cohort work is done only when the admitted-user list exists
- community setup is done only when the operating rule is written
- channel insight is done only when the memo exists

## Review Questions

Ask these in every review:

1. What shipped since the last check?
2. What proof exists?
3. What is blocked right now?
4. What is the next deliverable?
5. Is any lane drifting outside the wedge?

## Red Flags

Escalate immediately if any of these appear:

- homepage or outreach copy starts sounding like a generic habit app
- anyone starts leading with `blockchain`, `Fury`, `wager`, or `real money`
- creators are being contacted without a target list and fit logic
- practitioners are being pitched without the one-pager
- pages are built without source tracking
- users are admitted without cohort tags or source attribution

## Success Snapshot By Day 30

If the first 30 days are working, the board should show:

- one live no-contact landing page
- one live private-beta page
- one live emergency page or tool
- four published high-intent content pieces
- active creator outreach with real replies
- active practitioner outreach with real replies
- a first curated cohort admitted
- a clear view of the top two channels by source quality

## Bottom Line

This board exists to prevent drift.

Every lane should always be able to answer five things:

- what I own
- what I am doing this week
- what I ship next
- what is blocking me
- what proof shows the work is real
