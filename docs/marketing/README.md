# Growth & Marketing

Strategy, content plans, and user acquisition materials for Styx.

## What Lives Here

- **Go-to-market (GTM) strategy** — How we plan to reach our first 1,000 users
- **Content calendar** — Blog posts, social media, community engagement schedule
- **SEO strategy** — Target keywords, content topics, and search optimization for the No-Contact recovery niche
- **PR plan** — Press outreach for App Store launch and major milestones
- **Referral program design** — How existing users bring in new users
- **Partnership outreach** — Templates and strategy for therapist/coach partnerships

## Who Uses This

- **Non-technical co-founder**: This is your primary workspace. You own user acquisition, content, and partnerships.
- **Technical co-founder**: Understanding growth metrics, building features that support acquisition
- **Marketing agent (`styx-growth`)**: Drafts content, researches keywords, designs campaigns

## Key Metrics

- CAC (Customer Acquisition Cost)
- Organic vs. paid user split
- Referral conversion rate
- Content engagement (views, shares, sign-ups)
